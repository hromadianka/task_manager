# Task Manager Web

A web application for managing projects and tasks with features to:

- Create, edit, and delete projects
- Create, edit, delete, and mark tasks as done
- Set task deadlines and priorities
- Minimalistic UI using HTMX and Bootstrap

---

## Tech Stack

- Python 3.13  
- Django 5  
- HTMX  
- Bootstrap 5  
- Docker & Docker Compose
